# Our Stories Content

Place individual story markdown (`.md` or `.mdx`) files in this directory.

## Naming Convention
Each file should be named using the story slug, e.g. `how-healing-soil-began.md`.

## Frontmatter Template

```md
---
title: "How Healing Soil Began"
date: "2025-02-01"
excerpt: "The story behind our journey from Goa's red soil to handmade soap."
tags: ["origin", "goa", "founders"]
---

Your story content here...
```
