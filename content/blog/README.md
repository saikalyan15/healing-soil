# Blog Content

Place individual blog post markdown (`.md` or `.mdx`) files in this directory.

## Naming Convention
Each file should be named using the post slug, e.g. `why-natural-soap-matters.md`.

## Frontmatter Template
```md
---
title: "Why Natural Soap Matters"
date: "2025-01-15"
excerpt: "A short teaser sentence for the blog listing."
tags: ["natural", "soap", "goa"]
---

Your content here...
```
